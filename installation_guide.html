Installation Guide

  1. Get Python 3 from here.
  2. Check for dependencies from the command line:
    1. python -c "import requests"
    2. python -c "import tkinter"

    If any of these return ImportError, use pip to install them: pip install [package]

  3. Set up your sensorboard.

    If using the default sensorboard:

    1. Build the board based on the schematic.
    2. Install the Arduino IDE.
    3. Open sensorboard.ino in the Arduino IDE and edit it to set the wifi SSID and password.
    4. Upload it to the board using the IDE.
  4. Edit gui.py and set your sensorboards' IP addresses and ports.
  5. Run gui.py!
